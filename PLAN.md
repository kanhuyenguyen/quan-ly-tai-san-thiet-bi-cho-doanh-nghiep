# KẾ HOẠCH THỰC HIỆN ĐỀ TÀI

---
## Tên đề tài
Xây dựng hệ thống quản lý tài sản & thiết bị cho doanh nghiệp

---
## Mục tiêu
Xây dựng hệ thống quản lý tài sản và thiết bị nhằm hỗ trợ việc theo dõi và cập nhật dữ liệu trong doanh nghiệp

---
## Mô tả hệ thống
Hệ thống cho phép người dùng:
- admin
  + Quản lý thông tin tài sản và thiết bị
  + Thêm, sửa và xóa dữ liệu
  + Theo dõi tình trạng tài sản và thiết bị
  + Tìm kiếm thông tin 
  + Quản lý dữ liệu 
- user(nhân viên)
  + Xem thông tin tài sản và thiết bị
  + Tìm kiếm thông tin 
  + Gửi yêu cầu / báo cáo tình trạng thiết bị
  
---
## Yêu cầu hệ thống
### Yêu cầu chức năng
- Đăng nhập / phân quyền người dùng (Admin / User)
- Thêm / Sửa / Xóa tài sản
- Thêm / Sửa / Xóa thiết bị
- Tìm kiếm tài sản và thiết bị
- Theo dõi tình trạng tài sản và thiết bị
- Hiển thị danh sách dữ liệu
### Yêu cầu phi chức năng
- Giao diện đơn giản, dễ sử dụng
- Hệ thống hoạt động ổn định
- Dữ liệu được lưu trữ
  
---
## Công nghệ sử dụng
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: SQL Server
- Công cụ: GitHub, Visual Studio Code
  
---
## Kế hoạch thực hiện
### Tuần 1: Khảo sát & Phân tích yêu cầu
#### Nội dung cần thực hiện:
- Tìm hiểu yêu cầu đề tài
- Khảo sát + Xác định chức năng hệ thống
- Phân tích đối tượng sử dụng (Admin / User)
- Tạo repository trên GitHub
- Xây dựng Use Case cơ 
- Thiết kế cơ sở dữ liệu ban đầu
- Thiết kế sơ bộ giao diện cơ bản
#### Em thực hiện:
- Tìm hiểu + phân tích yêu cầu của hệ thống
- Xác định các chức năng chính cần xây dựng
- Thiết kế ý tưởng giao diện ban đầu
- Xác định cơ sở dự liệu ban đầu
#### AI hỗ trợ:
- Gợi ý thiết kế cấu trúc database và giao diện
- Hỗ trợ bố cục tài liệu
- Hỗ trợ xây dựng Use Case
- Hỗ trợ mô tả chức năng hệ thống
### Tuần 2: Thiết kế & lập trình hệ thống
#### Nội dung cần thực hiện:
- Bổ sung + hoàn thiện phần database
- Kết nối SQL Server với Python
- Hoàn thiện giao diện đăng nhập + giao diện quản lý tài sản và thiết bị
- Xây dựng chức năng đăng nhập/đăng xuất + phân quyền người dùng
- Xây dựng chức năng quản lý tài sản và thiết bị
- Xây dựng chức năng tìm kiếm và thống kê
- Kết nối database với hệ thống
#### Em thực hiện:
- Thiết kế giao diện hệ thống
- Thiết kế database
-  Viết và chỉnh sửa code các chức năng
- Kết nối database và kiểm tra chức năng
#### AI hỗ trợ:
- Gợi ý tối ưu giao diện và xử lý dữ liệu
- Hỗ trợ viết code 
- Hỗ trợ sửa lỗi
### Tuần 3: Kiểm thử & hoàn thiện hệ thống
